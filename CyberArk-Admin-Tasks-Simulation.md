# CyberArk Admin Tasks Simulation

## Daily Responsibilities of a CyberArk Administrator

### Account Onboarding

- Add privileged accounts into Vault
- Assign safe and platform
- Configure access permissions

### Password Rotation Monitoring

- Ensure CPM is rotating passwords
- Check for failed rotations
- Validate policy compliance

### Access Request Management

- Approve user access requests
- Monitor privileged session usage
- Enforce least privilege

### Session Monitoring

- Review PSM session recordings
- Detect suspicious admin activity
- Audit privileged actions

### Platform Management

- Configure password policies
- Manage account groups
- Update platform settings

### Incident Support

- Investigate unusual admin access
- Review session logs
- Support SOC investigations
